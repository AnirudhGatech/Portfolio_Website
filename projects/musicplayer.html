Analysis of Data Transfer and Encryption over Tor-Based Peer-to-Peer Connection versus Generic HTTPS and Configuring an HTTP Load Balancer with Auto Scaling

  • Tech Stack: Python, Nexmo, Node.js, Tor, Google Cloud Platform
  • Enhanced data transfer security by leveraging Python socket programming to bridge the gap between Tor and HTTPS, resulting in the implementation of 'torpy' and a Nexmo-integrated application for encrypted text communication within a Node.js environment.
  • Orchestrated HTTP load balancing with auto-scaling by configuring two managed instance groups as backends in US Central-1 and Europe West-1, complemented by the creation of a load balancer for VMs to showcase the capabilities of global load balancing and auto-scaling within the Google Cloud Platform.

Navigating Digital Battlefield During Russia-Ukraine Conflict - Project under Professor Nadiya Kostyuk, School of Public Policy and Cybersecurity Georgia Tech

Data Collection:

  • I used RIPE RIS historical routing data, focusing on RIBs dumped every 8 hours.
  • I selected three RIPE RIS route collectors in Amsterdam (RRC00), Moscow (RRC13), and Bucharest (RRC22).
  • I chose Moscow and Bucharest collectors for their proximity to Ukraine and local data collection, and Amsterdam for its global data collection due to its multihop nature.

Network Prefix Identification:

  • I identified network prefixes in Ukraine and Russia using the RIPE IP address allocations database.
  • I converted IPv4 address ranges to CIDR format for analysis.

BGP UPDATE Message Analysis:

  • I compared frequencies of BGP UPDATE messages for Ukrainian and Russian network prefixes from February 1 to April 30.
  • I conducted a detailed hourly analysis from February 23 to 28, noting significant peaks before the war's official start.

Geolocation and Heat Map:

  • I used MaxMind GeoIP2 database for geolocating IP addresses.
  • I plotted heat maps of BGP UPDATE message frequencies to visualize affected areas.

Prefix Origin AS Changes:

  • I analyzed changes in origin ASes for prefixes on a daily basis, noting rare events and unusual peaks.
  • I focused on prefixes in Ukraine, Russia, and Italy, examining the number of unique ASes leading to them.

Path Inflation Analysis:

  • I compared daily path inflation changes for Russian and Ukrainian IP networks.
  • I linked path inflation to global attitudes and policies against Russia, noting a continuous increase in path inflation for Russian networks.

Impact of Policies and Sanctions:

  • I discussed how routing policies, economic sanctions, IXP accessibility, and network fragmentation contribute to path inflation.

Subnet Announcements:

  • I monitored daily changes in subnet counts, linking significant increases to potential cyberattacks.
